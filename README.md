## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository. includes:

- app config
- base auth
- base test
- base mail service
- base jwt config
- base graphql(appolo driver)
- minimal prisma setup

## Installation

```bash
$ pnpm install
```

```bash
$ pnpx prisma generate
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
